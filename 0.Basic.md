# Basic

1. 驱动文件加载
```c
cp xxx.ko /rootfs/lib/moudles/5.4.31/
cd /rootfs/lib/moudles/5.4.31/
modprobe xxx.ko
lsmod
rmmod xxx.ko
lsmod
```
2. APP执行
```c
mknod /dev/xxx c major x //添加设备节点
./xxxApp /dev/xxx
```
3. 查看帮助文档
man n read      //n:"不同类型"
